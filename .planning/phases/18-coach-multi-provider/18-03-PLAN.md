| Boundary | Description |
|----------|-------------|
| app → user-configured endpoint | API key + chat content cross to an arbitrary host the user specifies |
| app → Keychain | Custom API key stored and read |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-18-07 | Tampering | User-configured base URL | mitigate | RemoteServerURLValidator enforces https:// for non-local hosts; http allowed only for localhost/.local/private IPs (V5 input validation) |
| T-18-08 | Information Disclosure | Custom API key | mitigate | Stored only in Keychain (com.goose.swift.custom-endpoint) with `kSecAttrAccessibleAfterFirstUnlockThisDeviceOnly`; baseURL/modelID (non-secret) in UserDefaults only |
| T-18-09 | Spoofing | Malicious/typo'd endpoint host | accept | Single-user app; user explicitly enters and owns the endpoint; URL is validated for scheme but host trust is the user's responsibility |
| T-18-SC | Tampering | npm/pip/cargo installs | mitigate | Zero external packages (D-08); no install tasks |
</threat_model>
